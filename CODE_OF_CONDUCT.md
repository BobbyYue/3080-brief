# Code of Conduct

We want this project to be a respectful, useful place for collaboration.

Participants are expected to be constructive, assume good intent, respect different backgrounds and levels of experience, and focus criticism on ideas and artifacts rather than people. Harassment, discriminatory language, threats, deliberate disclosure of private information, and sustained disruption are not acceptable.

Maintainers may edit or remove contributions that violate these expectations and may temporarily or permanently restrict participation when necessary. Report conduct concerns privately through the repository's maintainer contact or GitHub reporting tools rather than in a public issue.
